# UI Navigation & Theme

## 1. UI Design Principles
- **Theme:** Light Theme, White Background, Light Grey Surfaces.
- **Aesthetic:** Professional Government Dashboard, clean layout, modern design.
- **Typography:** Clear, highly legible sans-serif.
- **Exclusions:** No dark theme, neon colors, gaming, or futuristic AI styles.

## 2. State Admin Navigation
- **Dashboard:** High-level UP state analytics, District rankings.
- **State Heatmap:** Global UP view of hotspots.
- **Districts Overview:** List of all districts and their performance.
- **User Monitoring:** View all District Admins and system usage.
- **Reports:** Generate and export State-level reports.
- **Profile:** Settings, Change Password.

## 3. District Admin Navigation
- **Dashboard:** Today's violations, pending reviews, warning stats.
- **Traffic Officers:** Create, edit, transfer, or deactivate officers. Reset passwords.
- **Verification Queue:** Review 3+ warning cases (Recommend, Reject, Close).
- **District Heatmap:** Localized hotspot detection.
- **Reports:** District-specific daily/weekly/monthly reports.
- **Profile:** Settings, Change Password.

## 4. Traffic Officer Navigation (Mobile First)
- **Home/Dashboard:** Quick stats of own cases.
- **Capture Violation:** Main action button. Steps through Image, AI Review, Submission.
- **Search Vehicle:** Look up vehicle history by number plate.
- **My Cases:** History and status of captured cases.
- **Notifications:** In-app alerts for case status changes.
- **Profile:** Settings, Change Password.
